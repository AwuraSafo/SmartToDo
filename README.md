# Smart To-Do List
This is a Python task manager designed to help users organize tasks,
prioritize them, and keep track of deadlines. The project combines practical
coding skills with a human-centered approach to productivity.

## Features
- Add, view, edit, and remove tasks
- Assign priorities to tasks (1-5)
- Tasks are saved to a file for persistence between sessions
- Simple, clear interface for easy use

## Technologies Used
- Python 3
- File handling for data storage

## How to Run
1. Clone the repository
2. Run: `python todo.py`
3. Follow on-screen instructions to manage your tasks
